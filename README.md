# Game Review Hub

## サイト概要

### サイトテーマ
買い切りゲームを中心にレビュー・評価・情報共有ができるレビューサイトです。

### テーマを選んだ理由
近年のゲーム市場では、タイトル価格の上昇や大型作品の増加により、ユーザーが購入前に慎重な判断を求められる場面が増えています。私自身も買い切りゲームをよくプレイしますが、一本あたりの価格が高くなってきているため、「失敗したくない」という思いから購入前の情報収集が欠かせなくなっています。

しかし、買い切りゲームはレビューが分散しやすく、作品の良し悪しを判断するための情報が十分にまとまっていないという課題がありました。また、DLCの存在により「どこまでを買い切りと捉えるか」が曖昧になるケースもあります。

そのため本サービスでは、買い切りゲームを「本編購入のみで最後まで遊べる作品」と定義し、追加DLCの有無に左右されず主要体験が成立するタイトルを対象としています。これにより、ユーザーが安心して作品を比較・検討できる環境を整えたいと考えました。

ゲーム購入で失敗したくない人や、信頼できるユーザーの評価を参考にしたい人の助けとなるよう、買い切りゲームに特化したレビューサイトを制作することにしました。

### ターゲットユーザ
- ゲーム購入前に評価をしっかり確認したい人  
- ゲーム価格の高騰により購入を慎重に行う人  
- 自分に近いプレイスタイル・趣味のユーザーから情報を得たい人  
- 買い切りゲームが好きなプレイヤー  

### 主な利用シーン
- 購入前にゲームのレビューを確認したい時  
- 遊んだゲームの感想を投稿したい時  
- 気になるゲームの長所・短所を比較したい時  
- 自分と感性の合うユーザーを見つけたい時
